PHP Shortener
=============

Facilitates encoding/decoding of URLs with shortening services, such as 

* is.gd
* tinyurl.com
* bit.ly
* twurl.nl
* tr.im

Changelog 
=========

- 0.1: initial release
- 0.2: added digg.com. encode() now returns the passed URL if shortening failed. fetch() sends user agent
